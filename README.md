# ProteGear CLI

ProteGear CLI tools is an open source project to work with a CLI tool with ProteGear services and a GPS tracker named ALIVE from [ProteGear](https://protegear.com).

## Purpose

The primary functions planned for this project are:

* Connect to ProteGear services using an API key
* Extract data from GPS tracking
* Create and delete trips for well-known travels
* Export data (GPX files and more) for a given trip

The CLI tool is designed to handle a single ALIVE tracker or a list of trackers connected to one account.
